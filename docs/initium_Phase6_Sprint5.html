// Tickets du sprint

DEV-018

Suppression champ Autosave Interval

UI

Supprimer le champ "Autosave Interval (seconds)" et son label du popup Settings. Ce paramètre n'existe pas dans le backend et ne doit plus apparaître dans l'interface.

DEV-019

Settings : clic sur "Application Directories" ouvre le répertoire

UX

Dans le popup Settings, chaque entrée de la section "Application Directories" doit être cliquable et ouvrir le répertoire correspondant dans le gestionnaire de fichiers système (Nemo, Nautilus, Explorer selon la plateforme).

DEV-020

Settings : réduction de la taille des champs de saisie

UI

Revoir le CSS des champs de saisie du popup Settings pour les rendre plus compacts et lisibles — réduire padding, font-size et hauteur des inputs.

DEV-021

Add Launcher : "Choose file" ouvre ~/.config/initium/icons

UX

Dans le popup "Add Launcher", le bouton "Choose file" du champ "Icon (optional)" doit ouvrir directement le répertoire ~/.config/initium/icons au lieu du répertoire par défaut.

DEV-022

Réduction taille des launchers (×0.5)

UI

Réduire la taille des launchers de moitié afin de doubler le nombre de launchers visibles en largeur et en hauteur. Adapter la taille des icônes, labels et espacements en conséquence.

DEV-023

Launcher : clic sur l'icône lance l'application

UX

Le clic sur l'icône d'un launcher (et non sur un bouton séparé) doit déclencher le lancement de l'application ou l'ouverture du site web associé.

DEV-024

Export : ouvre l'explorateur dans ~/.config/initium

UX

Le clic sur le bouton "Export" doit ouvrir l'explorateur de fichiers directement dans ~/.config/initium pour permettre à l'utilisateur de récupérer facilement le fichier exporté.

DEV-025

Fenêtre redimensionnable jusqu'à la taille de l'écran

Config

Permettre à l'utilisateur d'étendre la fenêtre Initium jusqu'à la taille maximale de l'écran physique. Configurer les contraintes de taille dans tauri.conf.json en conséquence (supprimer ou augmenter maxWidth/maxHeight).

DEV-026

Localisation : Anglais / Français / Espagnol

i18n

Implémenter l'internationalisation (i18n) de l'application en anglais, français et espagnol. Rendre le sélecteur "Language" dans les Settings opérationnel — le changement de langue doit s'appliquer immédiatement à toute l'interface sans redémarrage.
